RetroRunner est un projet qui a été réaliser seul dans le cadre d'un projet en programmation de jeux video.
C'est un jeu de type 2D shooter Horizontal en javascript et avec l'utilisation de canvas. Le jeu est composé de 3 niveaux , dont le dernier est un combat de bose contre un personnage célèbre de l'animation japonaise "Dio Brando".

Dans le 1er niveau il faudra affronter une horde d'ennemie tous en esquivant les projectiles tirer par celle-ci.
Il est aussi possible d'utiliser une compétence spéciale pour figer les ennemies ce qui peut aider à esquiver plus facilement.

Dans le niveau 2 vous serez confronté à 2 machines qui vous tire dessus avec des rayons laser. Votre but sera de les esquiver jusqu'à la téléportation du personnage au niveau 3.

Le dernier niveau est le combat contre Dio Brando. Dans ce niveau nous avons une nouvelle arme et il faudra battre le boss qui a 3 phases de combat.
